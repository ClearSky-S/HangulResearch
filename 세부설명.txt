완성형 한글 2350자에 포함되는 음절 수 분석에 사용한 Python 프로그램에 대해서 알려주세요.

1. 파이썬이란?
파이썬(Python)은 비교적 배우기 쉬운 스크립트형 프로그래밍 언어입니다.
원래는 1991년 네덜란드 개발자인 귀도 반 로섬(Guido van Rossum)이 “누구나 쉽게 배울 수 있는 언어”를 목표로 만들었습니다. 현재는 데이터 분석, 웹 개발, 인공지능, 교육 등 다양한 분야에서 사용됩니다.
다른 언어보다 문법이 단순해 코드가 읽기 쉽고, 무료로 사용할 수 있으며, 필요한 기능을 추가할 수 있는 **다양한 라이브러리(도구 모음)**가 제공된다는 점이 특징입니다.
이러한 장점 때문에, 복잡한 설치 없이도 빠르게 한글 데이터를 분석할 수 있다는 점이 매력적이어서 본 연구에서도 파이썬을 사용했습니다.

2. 분석 과정
한글 글자의 사용 빈도를 파악하기 위해 먼저 분석할 자료를 텍스트 파일(.txt)로 정리했습니다.
그 후 파이썬으로 만든 프로그램을 이용해 파일을 읽고, 글자 및 어절을 분석해 빈도를 계산했습니다.
연구자가 손으로 일일이 세는 대신, 프로그램은 빠르게 총 글자 수, 각 글자의 사용 횟수, 특정 글자가 포함된 문장 등을 자동으로 찾아줍니다.
또한 워드클라우드(Word Cloud)라는 시각화 기법을 사용해 자주 사용된 어절을 한눈에 파악할 수 있도록 했습니다.

3. 워드 클라우드란?
워드 클라우드(Word Cloud)는 텍스트 데이터에서 자주 등장하는 단어를 시각적으로 표현하는 방법입니다.
단어의 크기나 색상으로 빈도를 나타내어, 어떤 단어가 많이 사용되었는지 쉽게 알 수 있습니다.
본 연구에서는 파이썬의 `wordcloud` 라이브러리를 사용해 워드 클라우드를 생성했습니다.

4. 한글 분석에 사용한 소스코드
아래는 실제로 사용한 파이썬 코드입니다.
코드의 역할을 간단히 설명하면 다음과 같습니다.
- input 폴더에 있는 텍스트 파일을 불러옴
- 글자 하나하나를 세어 사용 빈도를 기록
- ‘완성형 한글 2,350자’ 목록에 포함되는 글자만 따로 계산
- 목록에 없는 글자가 들어간 문장을 찾아 출력
- 본문에 사용된 어절의 빈도를 분석해서 출력
- 본문에 사용된 어절을 바탕으로 워드클라우드를 생성
- 최종 결과를 output 폴더에 저장
